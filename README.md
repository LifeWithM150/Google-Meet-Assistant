# Google-Meet-Assistant

    เพื่อช่วยตรวจสอบพฤติกรรมของผู้เรียน การปิด-เปิดกล้อง การทำกิจกรรมในระหว่างการเรียนผ่านกล้องถ่ายรูปที่ติดอยู่กับอุปกรณ์ที่นักเรียนใช้ในการเรียน
